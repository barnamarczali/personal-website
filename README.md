# Personal Website

Minimalist personal website for Barna Marczali.

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- React 18

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Design

- Background: `#063970`
- Accent: `#E58F65`
- Text: `#F1F1E7`
- Font: Manrope (sans-serif)
